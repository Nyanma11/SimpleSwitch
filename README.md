# SimpleSwitch

SimpleSwitchは、Windowsの外観モード（ライト/ダーク）を簡単に切り替えることができるアプリです。

## インストール

[こちらのページ](https://github.com/Nyanma11/SimpleSwitch/releases)からダウンロード後、解凍してお使いください。

### スタートアップへの登録

本アプリをOSのスタートアップに登録することで、PCの起動と同時に自動でアプリが立ち上がり、よりスムーズにご利用いただけます。手動で起動する手間を省きたい場合におすすめです。

1. `SimpleSwitch.exe`を右クリックし、「ショートカットの作成」をクリック。

2. Windowsキー + R を押して「ファイル名を指定して実行」を開く。

3. 以下の名前を入力し、「OK」をクリック。 

```Shell
shell:startup
```

4. 開いたフォルダに、作成したショートカットを入れる。

## 使い方

1. **アプリの起動**

   `SimpleSwitch.exe` をダブルクリックして起動します。
   >※ 起動後はタスクトレイ（画面右下の「^」の中）に隠れて常駐します。

3. **外観モードの切り替え**

   タスクトレイの中にあるSimpleSwitchのアイコンをクリックして、外観モードを切り替えてください。

4. **スケジュール機能の設定**

   タスクトレイの中のSimpleSwitchのアイコンを右クリックし、「設定」をクリックしてください。表示されたウィンドウから設定を行ってください。

6. **アプリの終了**

    タスクトレイの中にあるSimpleSwitchのアイコンを右クリックし、「終了」をクリックします。
